# 🛒 Market Management System

A console-based Market Management System developed using Java Core and Java Advanced.

## Features

- 🔐 User Registration & Login
- 👤 Admin & Customer Roles
- 📦 Product Management (CRUD)
- 🛍️ Purchase Products
- 📜 Purchase History
- 📊 Sales History
- 🔍 Filter Products by Price (Stream API)
- 💾 File Persistence

## Technologies

- Java Core
- Java Advanced
- OOP
- Stream API
- File I/O
- DTO Pattern
- Layered Architecture

controller
service
repository
entity
